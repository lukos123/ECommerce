Auth

POST /auth/register

Registers a new user.

Request

URL: /auth/register

Method: POST

Content-Type: application/json

{
    "username": "string",
    "email": "string",
    "password": "string"
}

Response

Status: 201 Created

Content-Type: application/json

{
    "access_token": "access_token",
    "refresh_token": "refresh_token"
}

POST /auth/login

Logs in a user.

Request

URL: /auth/login

Method: POST

Content-Type: application/json

{
    "username": "string", // or email
    "password": "string"
}

Response

Status: 200 OK

Content-Type: application/json

{
    "access_token": "access_token",
    "refresh_token": "refresh_token"
}

POST /auth/refresh

Refreshes the access token.

Request

URL: /auth/refresh

Method: POST

Headers:

Authorization: Bearer refresh_token

Response

Status: 200 OK

Content-Type: application/json

{
    "access_token": "access_token"
}
